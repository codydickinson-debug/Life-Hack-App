Help with Ascend — habits, money, and Cornileus

Ascend is open source and built by a small team. Below are the fastest ways to get help, report a bug, or send feedback.

Get help

💡 Try the in-app tour first

Open Settings → 🧭 Take the tour. 12 quick steps with live demo data show every feature in action.

✨ Ask Cornileus

Tap the ✨ button on any screen. Cornileus is the in-app counselor — it knows the app and can answer "where do I…?" questions about features as well as money questions.

📋 Open a GitHub issue

For bug reports, feature requests, or "this isn't working" — fastest path to a fix.

Found a vulnerability? Please report privately first — see our security.txt or open a private security advisory on GitHub.

Common questions

I forgot my passphrase. Can I recover my data?

No, by design. Encryption is zero-knowledge — the passphrase derives a key that's never stored or sent anywhere. There's no recovery code, no server backdoor. The only way forward is Settings → Reset everything, which wipes the encrypted blob and starts you over.

Always export a backup (Settings → Export data) BEFORE you set a passphrase, and again whenever you make big changes.

I cleared my browser data / uninstalled the PWA. Where's my data?

Local storage is gone with the browser data. If you exported a backup (.json or .ascend) it's still on your device — Settings → Import data restores it.

No remote copy exists; nothing leaves your device unless you opt into bank sync, AI counselor, or stocks queries.

How do I connect my bank?

Settings → Show advanced features → Bank sync. Requires a backend you control (Cloudflare Worker, free tier). Full walkthrough in DEPLOY.md.

Bank sync is read-only — Ascend never moves money.

Cornileus isn't answering. Why?

Most likely the backend AI proxy hit the per-user daily call cap (default 50). Try again tomorrow. If you've configured your own Cloudflare Worker with an Anthropic key, check the Worker logs.

Cornileus also goes silent if you close the chat sheet mid-request — that's intentional (chat auto-clears on close for privacy).

My data feels off — wrong totals, missing transactions, etc.

Settings → Export data first (preserve a snapshot), then check:

  • Privacy mode (👁 top-right) blurs amounts. If everything looks blurry, that's why.
  • Money sub-tabs filter the view — check you're on the right one.
  • Plaid sync can be delayed by hours for some banks. Settings → Bank sync → Sync now forces a refresh.
  • Hidden tabs/sections (Settings → Customize Today / Customize tabs) might be removing what you expected.

If something's still off, open an issue with the export attached (you can redact amounts first).

How do I install Ascend on my iPhone?

Open life-hack-app.vercel.app in Safari (not Chrome), tap Share → "Add to Home Screen". It'll launch fullscreen and feel like a native app.

An App Store build is in the works.

Can I sync across devices?

Not yet — Ascend is local-first by design (one of the trade-offs of having no accounts and no central server). Use Settings → Export data on one device, then Settings → Import data on the other. The encrypted export format includes everything.

How do I send feedback (not a bug, just an idea)?

Settings → Help → Send feedback in the app generates a pre-filled debug snapshot. Or open a GitHub issue tagged "enhancement".

What you can do without help

  • Export everything: Settings → Data → Export data (encrypted or plaintext).
  • Reset everything: Settings → Data → Reset everything (requires typing DELETE; backup-first prompt).
  • Replay the tour: Settings → 🧭 Take the tour.
  • Toggle privacy mode: 👁 icon in the top-right blurs every dollar amount.
  • Set a passphrase: Settings → Privacy & encryption → Set a passphrase. AES-encrypts everything at rest.
